## Project Overview

**Goal:** Create a city-builder with deep economic simulation, featuring:
- Large-scale maps (10km+)
- Complex production chains (25+ resources)
- Vehicle logistics (trucks, trains, conveyors)
- Citizen simulation with needs
- Soviet/brutalist aesthetic

**Tech Stack:**
- Godot 4.5+ (GDScript primary, C++ GDExtension for performance)
- Blender 4+ for 3D assets
- Linux-native development

**Timeline:** ~48 weeks (1 year)
